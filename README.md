Independent game studio building playable worlds that run right in the browser, plus tools and experiments for people who love MTG, Pokémon, Nintendo and classic MMORPGs. Most of what we build is open source.

## Flagships

- **[World of Aincrad](https://swordartonline.com.br/)** : browser MMORPG set in the Sword Art Online universe, 10+ years in the making.
- **[MU Browser](https://mu.beaterstudios.com/)** : the classic MU Online MMORPG, reimagined to run straight in the browser.

## Tools & apps

- **[Hisui Survey Corps](https://hisui.beaterstudios.com)** : field guide and tools for Pokémon Legends: Arceus.
- **[CatchBox](https://catchbox.vercel.app/)** : visual Pokédex tracker across every game, generation and box.
- **[PokEnergy](https://pokenergy.vercel.app/)** : virtual energy deck for the Pokémon TCG.
- **[MTG Horde Arena](https://mtg-horde-arena.vercel.app)** : simulator for Magic: The Gathering's Horde format.
- **[The Gathering Data](https://beater-studios.github.io/the-gathering-data/)** : interactive dashboard of 30+ years of Magic: The Gathering.
- **[NinWallet Sweep](https://ninwallet-sweep.onrender.com/)** : zero your Switch eShop wallet to unlock the region change.
- **[MU File Viewer](https://github.com/beater-studios/mu-file-viewer)** : web viewer for MU Online assets (textures, 3D models, audio, fonts).

## Prototypes & studies

Game jams, engine studies and early experiments: [OpenMU Browser Stack](https://github.com/beater-studios/openmu-browser-stack), [PokéMMO](https://github.com/beater-studios/pokemmo), [Snake Battle Arena](https://github.com/beater-studios/snake-battle-arena), [Cup Shuffle](https://github.com/beater-studios/cup-shuffle), and more across the repositories below.
